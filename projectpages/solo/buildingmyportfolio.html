Building My Portfolio

  • •
  • •

After completing my Masters degree, I tried out a free web development course through the website 'Code First Girls' which developed my interest in web design/ development. To build on my skills, I decided to try and build my own website to demonstrate what I had learnt. To start, I developed a design for my website by sketching out ideas and using Figma (a design tool) to create a clickable mockup of the website at different screen sizes. With this I was able to code my website using HTML, SASS and Javascript, referencing the mockup to check how I wanted to style the web pages.

In total the process took me about 2 months to complete, as I also coded a CV webpage linked to the website and there were many learning curves along the way.
